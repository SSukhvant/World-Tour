## üìã SEO Best Practices - Static Content Guidelines

### **Heading Hierarchy (Critical for SEO)**

‚úÖ **Correct Structure:**
```html
<h1>Main Page Title (Once per page)</h1>
<h2>Section Heading</h2>
<h3>Subsection</h3>
```

‚ùå **Incorrect:**
```html
<h1>Title 1</h1>
<h1>Title 2</h1> <!-- Multiple H1s confuse search engines -->
<h3>Skip H2</h3>  <!-- Don't skip levels -->
```

### **Image Optimization**

**With SEO (‚úÖ GOOD):**
```html
<img src="image.jpg" alt="Manali vacation package - mountain views and adventure">
```

**Without SEO (‚ùå BAD):**
```html
<img src="image.jpg" alt="">
<img src="image.jpg" alt="Image">
```

### **Alt Text Formula:**
```
[Keyword] [What's in the image] - [Context/Brand]
```

**Examples:**
- "Manali hill station package - scenic mountain landscape"
- "Maldives resort vacation - white sandy beach and turquoise water"
- "Travel insurance information - policy documents"

## ‚úçÔ∏è Content Writing for SEO

### **Title Best Practices:**

‚úÖ **Good Titles:**
- "Best Travel Packages to Manali - Adventure & Mountain Vacation"
- "Top 10 Destinations for Summer Holidays - Explore Now"
- "How to Plan Your Honeymoon - Complete Travel Guide 2025"

‚ùå **Bad Titles:**
- "Travel Packages"
- "Blog Post 1"
- "Click Here for Info"

### **Meta Description Best Practices:**

‚úÖ **Good (155 chars):**
"Explore Manali vacation packages with adventure activities, scenic views, and affordable prices. Book your mountain getaway with WorldTour4u today."

‚ùå **Bad:**
- "Travel packages" (too short)
- "This page contains information about travel packages and more..." (vague)
- No meta description (blank)

### **Keyword Placement:**

1. **H1 Title** - Main keyword (1st occurrence)
2. **First Paragraph** - Related keyword in first 100 words
3. **H2 Subheadings** - Natural keyword variations
4. **Body Content** - LSI keywords (related terms)
5. **Alt Text** - Descriptive with keyword
6. **Internal Links** - Use keyword as anchor text

### **Keyword Density:**

- Main keyword: 0.5-1%
- Related keywords: Natural placement
- Don't overstuff (avoid spammy look)
